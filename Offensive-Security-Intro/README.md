Offensive Security Intro — TryHackMe
What I Learned

Offensive security means thinking like a hacker to find weaknesses.

Websites can have hidden pages that attackers can discover.

Tools like dirb help find hidden URLs.

A hidden admin page can allow unauthorized actions.
